# flutter_tasks

깔끔한 **할 일 관리(ToDO)** 앱 입니다.
창업반 8기 개인과제 - TO DO APP UI 만들기

## 주요기능

**할 일 추가** - 제목과 상세내용을 입력해 새로운 할일 등록

**즐겨찾기** - 자주 사용하는 할일을 즐겨찾기 아이콘으로 등록 

**상세정보 추가** - 할일마다 별도의 상세내용 작성가능 

**다크모드 지원** - 라이트/다크 테마 자동전환
